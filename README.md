# 🗄 ShelbyVault

> Decentralized file storage dApp built on [Shelby Protocol](https://shelby.xyz)

![Shelby](https://img.shields.io/badge/Built%20on-Shelby%20Protocol-4fffb0?style=flat-square)
![Network](https://img.shields.io/badge/Network-Testnet-orange?style=flat-square)
![License](https://img.shields.io/badge/License-MIT-blue?style=flat-square)

ShelbyVault lets users upload, store, and retrieve files as blobs on the Shelby decentralized storage network — powered by Aptos blockchain and Jump Trading infrastructure.

## ✨ Features
- 📦 Upload files as blobs to Shelby network via RPC
- 🔍 Retrieve blobs by ID from any Shelby RPC endpoint
- 📂 Manage stored blobs — view expiry, copy IDs, open in explorer
- ⚙️ Configurable storage duration (7d → 1yr) and replication factor (1×–5×)
- 🔗 Aptos wallet integration for transaction signing

## 🛠 Tech Stack
- **Frontend:** HTML, CSS, Vanilla JS
- **Protocol:** Shelby Protocol — decentralized blob storage
- **Blockchain:** Aptos — coordination & settlement layer
- **Network:** Shelbynet Testnet
